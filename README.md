# Entendendo o Desenvolvimentismo Econômico

Este projeto utiliza o NotebookLM como ferramenta para resolver a confusão conceitual entre crescimento e desenvolvimento econômico, o que é desenvolvimentismo econômico, as evidências históricas e tecnológicas de que a prosperidade sustentável de uma nação depende de um Estado empreendedor capaz de liderar a transformação estrutural da indústria e assumir os riscos de inovações radicais.

O NotebookLM desenvolvido neste projeto pode ser acessado no link abaixo:

https://notebooklm.google.com/notebook/447c3258-8def-4dc6-9375-6def001ac92f

# Funcionalidades do Notebook
Resumos conceituais;
Mapas mentais;
Relatórios explicativos;
Cartões didáticos;
Testes conceituais;
Infográficos;
Resumo em vídeo. Para teste do notebook, é recomendado o uso de algumas das opções acima.

# Fontes
As respostas do NotebookLM são geradas exclusivamente a partir das fontes carregadas. Foram anexados:

Vídeos do youtube;

Relatórios institucionais.
